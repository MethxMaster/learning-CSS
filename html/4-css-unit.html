CSS Unit

Absolute unit

Symbol Explaination
px pixel is unit that most population used (1px = 0.75pt) it relate to the resolotion of the screen
pt point, 1 pt = 1/72 inchs it mostly use it printing field
cm centimetre
mm millimetre
in inches (1in = 96px = 2.54cm)
pc picas (1pc = 12pt)

Relative unit

Symbol Explaination
% Define size of the picture by using percent mostly use with height and width
em Reference to the element parent nearby. It is used to define the multiple value if the value was 10px
  • set 2em it will be 2em x 10px = 20px
  • set 1.4em it will be 1.4em x 10px = 14px
  • set 1.0em (Original size ) it will be 1.0em x 10px = 10px
rem Define size of the picture by refer to the root element. Normally font-size is 16px
vw 1% or 1/100 of viewport width
  • width of browser viewport = 750px, 1vw = 7.5px
vh 1% or 1/100 of viewport height
  • height of browser viewport = 900px, 1vh = 9px
vmin define to minimum value of viewport
vmax define to maximum value of viewport

Example

  
1. Absolute Unit



    HTML
        <p class="inch">text size 1 inch</p>
        <p class="cm">text size 1.5 cm.</p>
        <p class="mm">text size 5 mm.</p>
        <p class="pt">text size 10 pt</p>
        <p class="pc">text size 2 pc</p>




    CSS
        .inch{
            font-size : 1in ;
        }
        .cm{
            font-size : 1.5cm ;
        }
        .mm{
            font-size : 5mm ;
        }
        .pt{
            font-size : 10pt ;
        }
        .pc{
            font-size : 2pc ;
        }







    Output
    

        
text size 1 inch

        
text size 1.5 cm.

        
text size 5 mm.

        
text size 10 pt

        
text size 2 pc
