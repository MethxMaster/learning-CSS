Position

Introduction

Type Explanation
static default position
relative relate to the original position, position by off value
absolute position it on the coordinate refer to the to[ left corner (browser default 0,0)
fixed Fix position on viewport
sticky scroll until position limited

code


    CSS
    <style>
        .box{
            width: 100px;
            height: 100px;
        }
        #box1{
            background-color: orange;
            position: relative;
            left: 30px;
            top: 7px;
            z-index: 1;
        }
        #box2{
            background-color: greenyellow;
            position: absolute;
            left: 750px;
            top: 200px;
        }
        #box3{
            background-color: purple;
            position: fixed;
        }
        #box4{
            background-color: pink;
        }
        #box5{
            background-color: red;
            position: sticky;
            top: 0;
        }
        #box6{
            background-color: rgb(0, 167, 250);
            position: sticky;
            left: 700px;
            top: 0;
        }
    </style>
