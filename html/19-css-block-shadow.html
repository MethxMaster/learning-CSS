Block shadow

Introduction

This is structure of the "box-shadow"


    Sturcture
    box-shadow: x y blur spread color;

    Example
    box-shadow: 5px 5px 10px 5px gray;

More explanation,

  • x is shadow position (+ is right, - is left)
  • y is shadow position (+ is bottom, - is top)
  • blur is blur of shadow
  • spread is size of the shadow
  • color is color of the shadow
