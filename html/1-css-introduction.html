CSS Fundamental

Structure

  • Selector       is tag html, id or class 
  • Declaration    define property to selector
    
        selector{
            declaration
        }

Component

1. Selector is HTML tag, ID or Class, There are 3 types of Selector

  • Element selector is define style for html tag 
        
        h1{
            declaration
        }
  • Class selector is define style for using class. It's able to use more that one time. It can use it by put "." in fron of selector 
        
        .exam-class{
            declaration
        }
  • Element selector is define style for using ID. It's able to use only 1 time. It can use it by put "#" in fron of selector 
        
        #exam-id{
            declaration
        }

2. Declaration use to define value of selector

  • Property is attribute of selector.
  • Value is value of property.
    •     
        selector{
            property : value;
        }

Example

  
1. Tag selector

    

        HTML
            <p>MethxMaster</p>
    

    

        CSS
            p{
                color : red ;
                text-align : center ;
                padding : 10px ;
            }
    




    

        Output
        

            
developer01
  
2. Class selector

    

        HTML
            <h2 class="title">MethxMaster</h2>
            <p class="title">developer01</p>
    

    

        CSS
            .title{
                color : pink ;
                text-align : center ;
                padding : 10px ;
            }
    




    

        Output
        

            
MethxMaster

            
developer01
  
3. ID selector

    

        HTML
            <h2 id="title">MethxMaster</h2>
            <p id="message">developer01</p>
    

    

        CSS
            #message{
                color : green ;
                text-align : center ;
                padding : 10px ;
            }
    




    

        Output
        

            
MethxMaster

            
developer01
  
4. Union selector

    

        HTML
            <div id="box">
                <h2 class="title">MethxMaster</h2>
                <p id="message">developer01</p>
            </div>
    

    

        CSS
            .title, #message, #box{
                color : green ;
                text-align : center ;
                font-size : 18px ;
            }
    




    

        Output
        

            

                
MethxMaster

                
developer01
  
5. Tag.Class selector

    

        HTML
            <p class="error">MethxMaster</p>
            <p class="success">developer01</p>
    

    

        CSS
            p.error{
                color : red ;
            }
            p.success{
                color : green ;
            }
    




    

        Output
        

            

                
MethxMaster

                
developer01
