Font

Structure


    .message{
        font-family : Kanit ;
        font-size : 16px ;
        font-weight : bold ;
    }

Explanation

To work with font we can set font style, size weight and color

  • font-family we can set multi font in CSS file when the file render it will be chose font by order. 

    
    .message{
        font-family : Kanit, monospace ;
    }

  • font-size There are 2 ways to set font size.

    1. Set by const value 
      
    .message{
        font-size : x-large ;
    }
    2. Set by number value 
      
    .message{
        font-size : 16px ;
    }

  • font-weight

    1. Set by const value 
      
    .message{
        font-size : x-large ;
    }
    2. Set by number value 
      
    .message{
        font-size : 16px ;
    }
