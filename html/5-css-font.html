Font

Structure

CSS code

    .message{
        font-family : Kanit ;
        font-size : 16px ;
        font-weight : bold ;
        font-style : italic ;
    }
Output

    MethxMaster

Explanation

To work with font we can set font style, size weight and color as list below.

  • font-family we can set multi font in CSS file when the file render it will be chose font by order. 

    
    .message{
        font-family : Kanit, monospace ;
    }

    Firstly, We can use existing font which the browser support by checking this link
    https://www.cssfontstack.com/
    or we can use google font as this link below
    https://fonts.google.com/
    In case you would like to use google font you can follow this instruction.

    Here is an example
    
    MethxMaster

  • font-size define size of font (default = 16px) there are 2 ways to set font size.

    1. Set by const value ex. x-small, small, normal, large, ... 
      
    .message{
        font-size : x-large ;
    }
    2. Set by number value 
      
    .message{
        font-size : 16px ;
    }

  • font-weight define thickness of font. there are 2 ways to set font-weight

    1. Set by const value ex. lighter, bold, bolder, ... 
      
    .message{
        font-weight : bolder ;
    }
    2. Set by number value ex. 100, 200, ... , 900 
      
    .message{
        font-weight : 200 ;
    }

  • font-style define style of font. ex. italic, normal, oblique, inherit, ... 

    
    .message{
        font-style : italic ;
    }
