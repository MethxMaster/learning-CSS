Google font


Instruction

  1. Visit to this web site fonts.google.com

  2. Filter and search font which you would like to use in your work



    And then pick the one you like!

  3. You will see the page as picture below then look for this get code button and click it

  4. After that you choose way how to grab font code

    1. Reference link to the HTML file.
      Firstly, grab code from the internet.



      Then, paste code to the HTML file at element <head></head> and create embed style <style></style> 

      
    <!DOCTYPE html>                       
    <html>
        <head>
    
            <title>Title</title>

            <link rel="preconnect" href="https://fonts.googleapis.com">
            <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
            <link rel="https://fonts.googleapis.com/css2?family=Kanit:ital,wght@0,100;0,200;
            0,300;0,400;0,500;0,600;0,700;0,800;0,900;1,100;1,200;1,300;1,400;1,500;1,600;
            1,700;1,800;1,900&display=swap" rel="stylesheet">

            <style>

                .message{
                    font-family : Kanit ;
                }

            </style>

        </head>
        <body>
                
            <p class="message">MethxMaster</p>

        </body>
    </html>

      the output will be like

      
    MethxMaster

    2. Import google font link to the CSS file
      Firstly, grab code from the internet.



      Then, paste code to the HTML file at element <head></head> and create embed style <style></style>

      Create CSS file and import to the head of CSS file and then create the selector

      
    @import url('https://fonts.googleapis.com/css2?family=Kanit:ital,wght@0,100;0,200;0,300;0,400;0,500;0,600;0,700;0,800;0,900;1,100;1,200;1,300;1,400;1,500;1,600;1,700;1,800;1,900&display=swap')    

    .message{
        font-family : Kanit ;
    }

      then reference this CSS file that we created to HTML file

      
    <!DOCTYPE html>                       
    <html>
        <head>
    
            <title>Title</title>

            <link rel="stylesheet" href="style.css">

        </head>
        <body>
                
            <p class="message">MethxMaster</p>

        </body>
    </html>

      the output will be like

      
    MethxMaster
