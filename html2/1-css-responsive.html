Responsive Web Design

Introduction

It is a design for multi device. Devices has differnce screen size so the problem comes when you access diffent device. for example -> some content is be disabled


Resolution of device screen
Device Resolution
Mobile devices 320 - 480 pixels
iPads, Tablets 481 - 768 pixels
Labtops 769 - 1024 pixels
Desktops 1025 - 1200 pixels
TV, Widescreen 1201+ pixels

Meta tag

First thing that you need to do when you want to implement the responsive web page. You must decalre meta tag like example below. 


    <meta name="viewport" content="width=device-width, initial-scale=1.0">


Media Query

Media Query is style format to display output depends sizing of device's screen. 


    Concept

    @media (screen size){
        ... style ...
    }

    --------------------------
    code example
    
    /* smartphone size */
    @media only screen and (max-width:500px){
        #introduction{
            width: 100vw;
        }
    }

    /* iPad size */
    @media (min-width:501px) and (max-width:780px){
        #introduction{
            width: 50vw;
        }
    }

Example

code


    /* default setting */
    body{
        background-color: gray;
        color:white;
        text-align: center;
        padding: 100px;
    }
    h1{
        display: none;
    }

    /* smartphone setting */
    @media only screen and (max-width:500px){
        body{
            background-color: red;
        }
        #smartphone h1{
            display: block;
        }
    }

    /* tablet, ipad setting */
    @media (min-width:501px) and (max-width:768px){
        body{
            background-color: blue;
        }
        #tablets h1{
            display: block;
        }
    }

    /* desktop setting */
    @media (min-width:769px) and (max-width:1200px){
        body{
            background-color: green;
        }
        #normal h1{
            display: block;
        }
    }

    /* widescreen setting */
    @media (min-width:1201px){
        body{
            background-color: black;
        }
        #widescreen h1{
            display: block;
        }
    }
